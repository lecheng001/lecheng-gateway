# lecheng-gateway
lechenggateway,go,gateway,乐橙网关，GO语言，网关
乐橙网关分社区版和企业版
乐橙网关实现开箱即用，尽可能减少环境依赖。
乐橙网关配置数据采用json存储，linux以centos为例，不需要安装任何环境，即可运行乐橙网关

社区版功能列表：
1：控制台界面 全Web化配置
2：传输数据加密
3：协议适配
4：GET转跳协议
5：上传配置  HTTP文件上传
